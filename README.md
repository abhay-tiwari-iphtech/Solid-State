
# Solid State - Responsive Landing Page

A modern and fully responsive landing page built using **HTML5 and CSS3**.  
Inspired by the Solid State design, this project features angled section layouts, spotlight content areas, a project showcase grid, and a contact form.

## 🚀 Features

- Responsive design for all screen sizes
- Hero section with background image
- Spotlight sections with circular images
- Project grid layout
- Contact form with social links
- Clean and structured UI

## 🛠 Technologies Used

- HTML5
- CSS3
- Flexbox
- Media Queries

## 📱 Responsive Design

The layout adapts smoothly across:
- Desktop
- Tablet
- Mobile devices
